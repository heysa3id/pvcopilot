# PV Finance Model — GitHub Pages Ready

This package is a static web app prepared for direct upload to a GitHub repository and deployment with GitHub Pages.

## Included in this version
- Landing page
- Calculator section on the same site
- Grouped assumptions
- Eye / eye-off toggle for each assumptions card
- Calculated outputs: CAPEX, OPEX, load factor, LCOE, simple TRI/payback, and project TRI/IRR
- Summary panel listing only the model inputs

## Files
- `index.html`
- `styles.css`
- `script.js`
- `.github/workflows/deploy.yml`

## Deploy on GitHub Pages
1. Create a new GitHub repository.
2. Upload all files from this folder to the repository root.
3. Push to the `main` branch.
4. In GitHub, open **Settings > Pages**.
5. Under **Build and deployment**, choose **GitHub Actions**.
6. The included workflow will publish the website automatically.

## Local preview
You can open `index.html` directly in your browser, or serve the folder with any simple static server.
