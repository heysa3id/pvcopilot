PV Layout Estimator Estimate year-1 and 25-year net energy from site, weather, layout, and the full loss chain.

Overview & assumptions

This tool estimates energy production for a utility-scale PV system using hourly weather data, a detailed DC layout model, and a full loss chain to calculate year-1 and 25-year energy output.

Hourly irradiance transposition to plane-of-array. PVsyst-style thermal model with Uc/Uv. Linear or exponential degradation over the project lifetime.

Simulation methodology

POA = beam + diffuse + ground-reflected
Tcell = Tamb + POA / (Uc + Uv × WS)
Pdc = Pstc × (POA/1000) × (1 + γ × ΔT)

Loss chain: Soiling → IAM → Temperature → Quality → Mismatch → DC wiring → Inverter → Clipping → AC wiring → Availability
