# Documentation step images

Screenshots used by the Docs page for the LCOE Tool timeline. Add the following PNG files (captured from the running LCOE tool at `/lcoe-tool`) so the documentation displays real step images instead of placeholders.

| File | Content to capture |
|------|---------------------|
| `lcoe-step1-inputs.png` | Left panel: PDF upload zone and/or System tab; CAPEX tab with Save/Load Template buttons. |
| `lcoe-step2-finance-currency-status.png` | Finance tab visible; LCOE hero with currency badge; status badge (Excellent/Acceptable/Low) or thresholds popup. |
| `lcoe-step3-energy-cashflow.png` | Right panel: LCOE hero + KPI cards + chart area with Energy degradation and Cash flow tabs. |
| `lcoe-step4-download-report.png` | Export PDF button / download action (or report preview). |

Until these files exist, the Docs page shows the placeholder for each step.
